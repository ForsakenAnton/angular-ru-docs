Особенности и преимущества

Кроссплатформенная
Прогрессивные веб-приложения

Используйте возможности современной веб-платформы для создания приложений, похожих на нативные. Высокая производительность, автономная и нулевая установка.

Родной

Создание собственных мобильных приложений со стратегиями из Cordova, Ionic или NativeScript.

Рабочий стол

Создавайте приложения для настольных компьютеров в Mac, Windows и Linux, используя те же методы Angular, которые вы изучили для Интернета, а также возможность доступа к собственным API-интерфейсам ОС.

Скорость и производительность
Генерация кода

Angular превращает ваши шаблоны в код, который высоко оптимизирован для современных виртуальных машин JavaScript, предоставляя вам все преимущества рукописного кода и производительность инфраструктуры.

Универсальный

Подайте первое представление вашего приложения на Node.js®, .NET, PHP и других серверах для почти мгновенного рендеринга только в HTML и CSS. Также открывает путь для сайтов, которые оптимизируют для SEO.

Разделение кода

Приложения Angular загружаются быстро с новым маршрутизатором компонентов, который обеспечивает автоматическое разделение кода, поэтому пользователи загружают только код, необходимый для отображения запрашиваемого представления.

Производительность
Шаблоны

Быстрое создание представлений пользовательского интерфейса с простым и мощным синтаксисом шаблона.

Угловой интерфейс командной строки

Инструменты командной строки: начните строить быстро, добавьте компоненты и тесты, а затем немедленно разверните.

IDE

Получите интеллектуальное завершение кода, мгновенные ошибки и другие отзывы в популярных редакторах и IDE.

Полная история разработки
Тестирование

С помощью Karma для модульных тестов вы можете знать, что вы ломали вещи при каждом сохранении. Кроме того, Protractor позволяет выполнять ваши сценарии быстрее и стабильнее.

Анимация

Создавайте высокопроизводительные, сложные хореографии и графики времени анимации с очень небольшим количеством кода с помощью интуитивно понятного API Angular.

Специальные возможности

Создание доступных приложений с компонентами с поддержкой ARIA, руководствами для разработчиков и встроенной тестовой инфраструктурой a11y.
